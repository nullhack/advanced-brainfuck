The sequence of initialization is as follows:
- if there's an input code, run it
- after running the input the interpreter enter in 
  shell mode keeping values.
- an empty value finish the interpreter.

Command line execution is simple:

>>> from brainfuck import BrainFuck
>>> bf = BrainFuck()
>>> bf.execute('+++.')
3
>>> bf.execute('[-]')
>>> bf.execute('{bflib/plus_ten}*')
importing: bflib/plus_ten
|10|
>>> bf.execute('&')
19 +++.[-]++++++++++*&
